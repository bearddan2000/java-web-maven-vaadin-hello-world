# java-web-maven-vaadin-hello-world

## Description
A POC for vaadin UI.

## Tech stack
- java
- maven
  - vaadin

## Docker stack
- adoptopenjdk/maven-openjdk11

## To run
`sudo ./install.sh -u`
http://localhost

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
